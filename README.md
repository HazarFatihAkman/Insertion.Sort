## Patika.dev Selection Sort Project

> Example Array : [22, 27, 16, 2, 18, 6]

### Stages

* n   : 22, 27, 16, 2, 18, 6
* n-1 : 2, 27, 16, 22, 18, 6
* n-2 : 2, 6, 16, 22, 18, 27
* n-3 : 2, 6, 16, 18, 22, 27

### Big-O

> Best Case    : O(n^2)
>
> Average Case : O(n^2)
>
> Worst Case   : O(n^2)
